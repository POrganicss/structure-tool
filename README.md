# StructureTool

代码介绍
该代码主要用于实现以下功能：
  1.根据分子骨架定向生成小分子库。
  2.辅助Gaussian进行机理计算。
  3.进行自动分子对接。（待开发）
  4.进行自动Gromacs分子动力学模拟。（待开发）
